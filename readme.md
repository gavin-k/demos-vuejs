Vue 实例的配置选项: 
	el : 在哪个选定的元素上.
	data : 自定义数据.
	computed : 计算属性.
	methods : 自定义方法.
	components : 组件.
	filters : 自定义的过滤器.
	directives : 自定义的指令.
	elementDirectives : 自定义的元素指令.
